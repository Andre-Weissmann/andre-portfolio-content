# andre-portfolio-content

Version-controlled content layer for [andre-weissmann-portfolio.pplx.app](https://andre-weissmann-portfolio.pplx.app).

This repo is the **content layer** -- raw project data, case studies, and assets. The application that serves it is in [`andre-portfolio-cms`](https://github.com/Andre-Weissmann/andre-portfolio-cms).

## What this repo contains

- `projects.json` -- all portfolio project metadata (titles, descriptions, tools, findings)
- `about.json` -- bio and skills section content
- `Bike_Sales_Dashboard.xlsx` -- Excel project source file
- `DataProfessionals_Survey.pbix` -- Power BI project source file

## Projects in the portfolio

1. Nashville Housing Data Cleaning (SQL)
2. BMI and Waist-to-Hip Risk Screening (Python)
3. Power BI Data Professional Survey (Power BI)
4. Seattle Airbnb Pricing Analysis (Tableau)
5. Bike Sales Dashboard (Excel)

## Application layer

The CMS and server that reads this content lives in [`andre-portfolio-cms`](https://github.com/Andre-Weissmann/andre-portfolio-cms).
